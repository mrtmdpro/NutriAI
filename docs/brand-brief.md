# NutriAI brand brief

The single source of truth for visual identity. Everything in
`scripts/stitch-generate.ts` reads from this brief; theme tokens in
[`app/globals.css`](../app/globals.css) and constants in
[`lib/theme.ts`](../lib/theme.ts) implement it.

## Positioning

- **Product**: Evidence-based nutritional intelligence platform.
- **Audience**: General consumers in Vietnam (vi default) + English.
- **Voice**: Clinical-but-friendly, scientifically rigorous, accessible.
  Never lifestyle-influencer, never pharma-clinical.
- **Differentiation**: Every claim cites a source. Every supplement has a
  Quality Index. The AI never hallucinates.

## Color

| ---------------- | ---------------------- | ------------- |
| Primary (mint)   | oklch(0.72 0.16 152)   | ~#86EFAC      |
| Foreground       | oklch(0.18 0.02 152)   | ~#0c1f15      |
| Background       | oklch(1 0 0)           | #FFFFFF       |
| Accent (wash)    | oklch(0.94 0.05 152)   | ~#D5F0DD      |
| Muted            | oklch(0.97 0.01 152)   | ~#F4F8F5      |

Hue is fixed at **152** across the palette. Do not introduce other hues
except for charts (sequential green ramp).

## Typography

- Sans: **Geist** (Geist Sans on Google Fonts)
- Mono: **Geist Mono** (timestamps, IDs, payment codes)
- Heading rhythm: 600 weight, tight tracking
- Body rhythm: 400, generous line-height

## Logo brief

The mark must read as **"a leaf-shaped synapse"** — a single organic
silhouette suggesting both nutrition (leaf) and reasoning (a connecting
node or vein pattern). One ink, no gradients. Geometric, not illustrative.

### Constraints

- One color (primary green) on a clean surface; reverse permitted for dark.
- Optical-balance the leaf so it works at 16px favicon size **and** 1200×630
  OG hero.
- The "tip" of the leaf should subtly suggest a circuit node or signal
  point — restrained, not literal.
- No gradients, no glow, no rainbow, no spirals.
- Avoid: medical cross, pill bottle, mortar & pestle, DNA helix,
  flowers, ribbons, monogram inside circle.
- Wordmark: "NutriAI" set in Geist Sans semibold. The "AI" pair gets a
  subtle weight or tracking shift — not a different color — so the
  AI angle reads without breaking the mark's monolithic feel.

## Page inventory

The Stitch generation script targets every distinct UI state below.
Each line is a single Stitch generation (1 of the 350/month quota).
Numbering matches the slug used on disk.

### Brand
- `00-logo-system` — brand sheet showing all logo variants on a grid.

### Public BR1 — Knowledge Hub
- `01-landing` — hero search + value props + Quality Index preview.
- `02-search-results` — populated query, filter chips, supplement + ingredient sections.
- `03-search-empty` — Knowledge-Hub-loading state (first launch).
- `04-supplement-detail-ingredients` — header + tabs + ingredients table.
- `05-supplement-detail-evidence` — same header, Evidence tab active with cited cards.
- `06-supplement-detail-quality` — Quality tab with score breakdown bars.
- `07-quality-index` — tier S/A/B/C ranked grid.
- `08-feed` — articles + KOL videos + personalized rail.

### Auth
- `09-login` — email/password + Google OAuth.
- `10-register` — same shell, "create account" copy + check-inbox state.

### BR2 — Adherence
- `11-dashboard-today` — today's regimen with intake toggles + streak.
- `12-dashboard-empty` — first-time empty state with "create regimen" CTA.
- `13-regimen-list` — regimens grid with edit affordances.
- `14-regimen-builder` — multi-item form with day/time chips + push toggle.
- `15-analytics` — adherence stats + 12-week heatmap + line trend.

### BR4 — RAG chat
- `16-chat-empty` — welcome card with suggested prompts.
- `17-chat-conversation` — user + assistant turns with inline `[c1]` citation chips.

### Monetization
- `18-pricing` — Free vs Pro tiers, feature table, SePay attribution.
- `19-billing-pending` — VietQR + amount + memo + bank details + status pulse.
- `20-billing-active` — active Pro with renewal date + history table.

### System
- `21-mobile-nav` — left-sheet hamburger drawer with nav links + auth CTAs.
- `22-not-found` — bilingual 404 surface inside the app shell.

22 generations + the brand sheet = 23 total. Well under the 350/month
free cap with room for revisions.

## How to regenerate

```bash
npm run stitch:generate
```

The script is idempotent — it skips slugs that already have
`design/stitch/<slug>.{html,png}` on disk. Delete the output file to
re-roll a single screen.
